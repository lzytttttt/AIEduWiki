# 自适应学习系统

## 定义

自适应学习系统利用 AI 根据每位学习者的表现、行为和实时理解程度，动态调整教学内容和学习节奏——突破传统"一刀切"的授课模式。

## 两大范式

### 静态适应（旧范式）
- 预设难度的测验题
- 所有学习者统一学习节奏
- 通用反馈，无差异化
- 对学习者认知变化响应极为有限

### 动态/实时适应（当前前沿）
- 学习过程中持续分析学习者答题反应
- 根据表现实时调整题目难度
- 多模态内容分析（视频 + 音频 + 文本）
- 个性化反馈与针对性举例
- 课程结束时生成个性化学习总结

## 新进展：GraphMASAL（arXiv:2511.10000）

Graph-based Multi-Agent System for Adaptive Learning，2025年提出的图增强多智能体自适应学习框架：

### 三大组件

| 组件 | 功能 |
|------|------|
| **动态知识图谱** | 持久化、有状态的学习者建模 |
| **三智能体编排**（Diagnostician + Planner + Tutor） | LangGraph 协调的诊断-规划-教学闭环 |
| **知识图谱检索** | 双编码器密集检索 + 交叉编码器列表重排 |

### 关键创新
- **有状态建模**：知识图谱持久化追踪学习者历史，而非仅依赖当前 session
- **可解释诊断**：Diagnostician 输出诊断理由，而非黑盒预测
- **知识图谱锚定**：所有内容选择和生成都基于 KG，而非自由生成

## 技术组件

1. **学习者建模** — 追踪学习者知识状态的动态变化（见 [知识追踪](../concepts/知识追踪/)）
2. **内容个性化** — 根据学习者特征定制题目/材料
3. **测评自适应** — 动态调整题目难度
4. **反馈生成** — 个性化、及时、有建设性
