# 知识追踪

## 定义

知识追踪（Knowledge Tracing, KT）旨在根据学习者过往的学习交互记录（如答题情况、提示使用、学习时长），建模其知识状态的动态演化，从而预测在后续任务上的表现。

## 传统方法的问题

| 方法 | 问题 |
|------|------|
| 深度学习（DL）方法 | 依赖特定平台标识符；隐表示不透明；无法跨平台迁移 |
| LLM Prompting | 容易产生无根据的知识状态幻觉 |
| LLM Fine-tuning | 高度领域依赖；适配新领域成本高 |

## 2026 年新方案

### RAG-KT（检索增强知识追踪，arXiv:2604.10960）
- 跨平台泛化能力强，以"问题群"抽象统一异构平台数据表征
- 多源结构化上下文 + 跨源对齐，为每次预测检索可靠上下文
- **实验结果**：三个公开 KT 基准数据集上一致的准确率和鲁棒性提升
- 在跨平台（分布偏移）条件下仍保持准确

### Responsible-DKT（负责任的深度知识追踪，arXiv:2604.08263）
神经符号融合路线，发表于 AIED 2026：

| 成果 | 数据 |
|------|------|
| 仅 10% 训练数据达到 0.80 AUC | 10% → 0.80 |
| 全量数据达 0.90 AUC | 提升 13% |
| 早期序列预测误差低 | 时序可靠性提升 |
| 预测不一致率最低 | 序列长度横向上下对齐 |

关键洞察：**重复错误（非掌握信号）对预测更新影响最强**——这为教育干预提供了可操作靶点。

### 神经符号融合的核心价值
1. **可解释性**：可解释计算图暴露每个预测背后的逻辑，支持本地和全局解释
2. **数据效率**：符号规则弥补数据不足，10% 数据即可用
3. **时序一致性**：预测更新方向与学生实际反应随时间保持对齐
4. **教育假设验证**：可实证评估"非掌握规则是否真的影响预测"——打开了教育研究与 ML 的双向通道

## 核心指标

- **AUC**（曲线下面积）— 预测准确率
- **时序一致性** — 预测随时间变化的稳定性
- **跨平台泛化** — 在新平台/分布上的表现
