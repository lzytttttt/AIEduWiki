# 教学智能体

## 定义

教学智能体（Pedagogical Agent）是承担教师、导师或学习伙伴角色的 AI 系统——通过与学生对话，交付、调整或支持教学内容。

## LLM 时代的教学智能体（2022–2025）

自 GPT-3.5/4 出现（2022年底）以来，LLM 为教学智能体带来了前所未有的能力：
- 跨学科自然语言理解
- 复杂推理能力
- 自适应对话

## 设计维度（综述研究归纳）

| 维度 | 选项 |
|------|------|
| 交互方式 | 反应式（被动应答）vs 主动式（主动发起学习行为） |
| 领域范围 | 领域专用 vs 通用途 |
| 角色复杂度 | 单角色 vs 多角色（多智能体协作） |
| 系统集成 | 独立运行 vs 与其他系统集成（LMS、VR等） |

## 新兴趋势（2025-2026）

基于 arXiv:2604.12253 的系统综述（52篇研究，2022.11-2025.01），四条新兴方向：

### 1. 多智能体仿真
多个专业化智能体协作，各自负责一个评估维度。典型架构：诊断智能体 + 规划智能体 + 教学智能体（如 GraphMASAL 三智能体架构）。

### 2. 虚拟学习者仿真
用 LLM 模拟学习者行为，用于规模化评估教学智能体——无需招募真实学习者即可做系统测试。

### 3. 可解释教学编排
将单体黑盒 LLM 替换为特化模型集成，每个教学决策都有显式理由输出，教师可逐一审核（AIED 2026）。

### 4. 具身智能 + LLM
VR/AR 场景中的教学智能体，将空间交互与 LLM 语言能力结合。

## 多智能体教学系统

现代系统采用**多个专业化智能体**协作：
- 每个智能体负责一个评估维度（如：数学准确性、可读性、真实性）
- 可按学科、角色（教师 vs 评估者）或模态进一步分工

## 前沿趋势

1. **多智能体仿真** — 模拟真实学习环境
2. **虚拟学习者仿真** — 规模化评估教学智能体
3. **VR/AR + LLM 智能体** — 沉浸式学习
4. **学习分析集成** — 数据驱动的自适应
5. **可解释编排** — 打破黑盒，提升教师信任
