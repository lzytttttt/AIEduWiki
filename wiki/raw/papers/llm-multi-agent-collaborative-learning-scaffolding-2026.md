# LLM Multi-Agent Simulation for Collaborative Learning Scaffolding

**arXiv:** 2604.11161v1 | **Published:** 2026-04-13 | **Cats:** cs.HC, cs.MA
**Authors:** Han Wua, Lishan Zhang, Chunming Lu
**PDF:** https://arxiv.pdf/2604.11161v1

## Abstract

This study proposes an LLM-based multi-agent simulation approach to investigate collaborative learning processes and the effectiveness of instructional scaffolds prior to actual classroom deployment. The simulation system was implemented using the MetaGPT framework and GPT-4o, comprising one teacher agent and five distinct student roles (Leader, Supporter, Expounder, Rebutter, and Summarizer). Two scaffolding strategies, "Deep Think before Speak" and "Direct Speak", were compared across ten classical Chinese poetry appreciation tasks.

## System Architecture

- **Framework:** MetaGPT + GPT-4o
- **Agents:** 1 teacher + 5 student roles (Leader, Supporter, Expounder, Rebutter, Summarizer)
- **Task:** 中国古典诗歌鉴赏
- **Evaluation:**话语质量 + 行为分析

## Key Findings

"Deep Think before Speak" 脚手架策略显著改善：
- **话语多样性** ↑（话语更丰富多元）
- **交互深度** ↑（从"Active"参与提升到"Constructive"和"Interactive"知识共建）
- **内容重复** ↓

符合 ICAP 框架：脚手架将学生从简单"Active"参与推向"Constructive"和"Interactive"的知识共建。

## 意义

LLM 多智能体系统可以**在真实教室部署前**有效模拟协作学习过程和脚手架效果——大幅降低研究成本。
